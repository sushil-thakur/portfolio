AI Assistant

For my AI Assistant project, I developed a voice recognition system that allows users to interact and give commands through speech. The assistant is built using JavaScript for functionality, alongside HTML and CSS for the user interface. It integrates APIs to enhance responsiveness and provide a seamless user experience.
